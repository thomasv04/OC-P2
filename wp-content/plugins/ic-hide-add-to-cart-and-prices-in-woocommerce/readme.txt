IC Hide Add to Cart and prices is useful if you want to hide add to cart buttons and prices in WooCommerce for products in your store or by products categories. 


== Description ==

This plugin is an extension of WooCommerce and provide a settings panel where you can easily disable add to cart buttons and prices for product in your store or specific products 
categories. In the options you can disable add to cart buttons and prices only for the first level of category, no sub categories will be disabled. You can also display a personalized text when prices are hidden.  

= Features: =
* Disable WooCommerce Add to Cart buttons for simple product.
* Disable WooCommerce Add to Cart by first level simple product categories.
* Hide WooCommerce prices for simple product.
* Hide WooCommerce prices by product categories.
* Insert custom text in prices field when prices are hidden.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. When the plugin is activated the settings are available in the WooCommerce menu > Settings > HACP Plugin Tab. 




== Frequently Asked Questions ==

= Where are the settings of the plugins? =

The settings of the plugin are in the WooCommerce menu > Settings > HACP Plugin Tab.

= Are "Add to Cart" buttons and prices hidden also in the shortcodes provided by WooCommerce? =

Yes, if the options are activated the products "Add to Cart" and prices will be hidden in the shortcodes.

= Is the plugin tested for WordPress Multisite ? =

No, the plugin is not yet tested for Multisite, it only works in single WordPress configuration.

= Can I disable add to cart and prices for subcategories products ? =

No, the plugin works only for first level category product.

= The plugin works with variable products and variations ? =

No, the plugin works only with simple products, it's not yet tested for variable products and variations.
